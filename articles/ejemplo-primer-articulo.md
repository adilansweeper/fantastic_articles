## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que te permite formatear texto de manera sencilla. Algunos elementos básicos incluyen:

### Formato de Texto

- **Negrita**: Usa `**texto**` o `__texto__`
- *Cursiva*: Usa `*texto*` o `_texto_`
- `Código`: Usa comillas invertidas `` `código` ``

### Listas

Puedes crear listas ordenadas y desordenadas:

1. Primer elemento
2. Segundo elemento
3. Tercer elemento

- Elemento A
- Elemento B
- Elemento C

### Enlaces e Imágenes

- Enlaces: `[texto del enlace](URL)`
- Imágenes: `![texto alternativo](URL de la imagen)`

## Bloques de Código

Puedes incluir bloques de código con sintaxis específica:

```python
def saludar(nombre):
    return f"¡Hola, {nombre}!"

print(saludar("Mundo"))
```

```javascript
function saludar(nombre) {
    return `¡Hola, ${nombre}!`;
}

console.log(saludar("Mundo"));
```
